前端入门之CSS居中总结

Posted on 2018-05-08

CSS布局中，居中是最常见的，有的时候也是最困扰的，今天我把居中的主要实现方法总结如下：

1.水平居中

1.1 内联元素水平居中

在父级块类元素中，写入：.center-children { text-align: center; }可实现居中

1.2 块级元素居中

给块级元素margin-left 和 margin-right 值为auto 。这时要设置width，否则块级元素的width会默认继承父级元素的宽度，居中就没有意义了，简写为：

.center-me { margin: 0 auto; }

1.3 多个块级元素居中

可以给多个块级元素设置display:inline-block;同时父级设置text-align:center;就可以实现。

2.垂直居中

2.1内联元素居中

2.1.1单个内联元素居中

    第一种方法：同时设置padding-top及padding-bottom相等。第二种方法：设置height值与line-height值，并让两者相等。

2.1.2多个内联元素居中

    可以用display:table来实现。

或者用flex弹性布局：

.flex-center-vertically { display: flex; justify-content: center; flex-direction: column; height: 400px; }

2.2块级元素居中

2.2.1知道块级元素高度的时候，可以通过以下代码：

    .parent { position: relative; }

    .child { position: absolute; top: 50%; height: 100px; margin-top: -50px;  }

2.2.2不知道高度的时候，可以通过以下代码：

    .parent { position: relative; }

    .child { position: absolute; top: 50%; transform: translateY(-50%); }

2.2.3用flex布局来实现

    .parent { display: flex; flex-direction: column; justify-content: center; }

3.水平及垂直居中

3.1 固定宽高时，用定位来实现。

.parent { position: relative; }

.child { width: 300px; height: 100px; padding: 20px; position: absolute; top: 50%; left: 50%; margin: -70px 0 0 -170px; }

3.2 不定宽高时，用定位来实现

.parent { position: relative; }

.child { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }

3.3 flex布局来实现

.parent { display: flex; justify-content: center; align-items: center; }

3.4 grid布局来实现

body, html { height: 100%; display: grid; }

span { / thing to center / margin: auto; }

以上内容就是居中的方法，基本上涵盖了所有情况。
