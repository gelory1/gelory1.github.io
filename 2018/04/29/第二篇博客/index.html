第二篇博客

软件安装总结
1.lunix虚拟机的安装
深度Linux镜像文件下载速度很慢，我先把教程看了一遍，可以将系统放在虚拟机上或者双系统。lunix主要通过命令行操作。lunix系统的学习使用能够让我们习惯和掌握部分命令行的操作。
2.VSCode编辑器
一种方便的轻量级代码编辑器，打开速度快，支持Emmet快速编写代码，支持git操作。
Emmet快捷键列表：https://docs.emmet.io/cheat-sheet/
3.git bash
git bash安装完成后，通过相关命令行可以对相应的目录及文件进行操作，本地修改后的文件可以push到github仓库上，是一种先进的分布式版本控制系统。
4.FQ
之前安装过蓝灯软件，但是一直不太稳定，今天安装了phiphon，目前使用没什么问题，如果出现问题再换收费的软件。
5.github使用
新建仓库后，需要在git bash运行命令（虽然我不明白这些命令都是什么意思），然后设置SSH keys，设置之后就可以将通过编辑器修改的文件push到刚才新建的库里了。通过库里面的settings设置githubpages，就可以将html文本直接通过网址打开了，展示起来很方便。
